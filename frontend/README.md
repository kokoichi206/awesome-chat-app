# Awesome-Chat-app_フロントエンド

## 基本構成
* Vue3 + Vuetify + Pinia を基本構成としている

## セットアップ
```sh
$ yarn
$ yarn build
```

## Compile and Hot-Reload for Development

```sh
$ yarn preview
```

### Type-Check, Compile and Minify for Production

```sh
$ yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
$ yarn lint
```

### test

```sh
$ yarn ts-jest  
```
